    An environment for your students

    Prog1Box provides an easy and clean computer system environment.
    Easy to install and easy to use. It is a live CD image that you boot and can configure online.

    Why...?

    Computer laboratories may provide a initial computer system that tries to be good for all students. Often you want a enviroment where students can be productive and free of distractions. It needs to provide right tools and the right configuration so they can be productive. Prog1Box provide such system: it is a live CD that is easy to configure and adapt based upon your needs.

    Want to install new software? Done. Want to configure the system firewall? Done. Want to change some files? Done.

    How...?

    You first need to contact our team to ask for a Prog1Box admin account. Send information about your institution and network. Then you just need to follow three easy steps:

    • Configure your system here (apps, firewall)
    • Download our live CD
    • Run it!

    When a computer from your network starts, it will ask our servers for the defined configuration and adjust itself. It is that simple.
